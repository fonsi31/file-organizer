# File Organizer

A Python script that organizes files into folders based on file type

## Features
- Sorts files by extension
- Automatically creates folders
- Handles duplicate filenames safely

## Categories
- Images
- Documents
- Videos
- Audio
- Archives
- Code
- Executables
- Others

## How to Run

```bash
python file_organizer.py
```

## Example

Before:
```text
test_folder/
  image.jpg
  notes.pdf
```

After:
```text
test_folder/
  Images/
    image.jpg

  Documents/
    notes.pdf
```
